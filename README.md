## 📌 Project Overview
"Genomics for All" is an open-source, public health educational platform designed to democratize access to information regarding genomic medicine. Developed as a Software Engineering University Extension Project (PEX) for Descomplica Faculdade Digital, this hub is tailored specifically for the community of Gilfach Goch, South Wales. 

The project bridges the gap between advanced clinical genetics and community health literacy, outlining how the **All Wales Medical Genomics Service (AWMGS)** utilizes genetic sequencing for precision medicine, rare disease diagnosis, and advanced cancer care. It also serves as an academic bridge to advanced research topics, notably the genomic risk factors associated with Cardiovascular and Hematological conditions (such as Thrombophilia).

## 🎯 Objectives
1. **Public Health Impact:** Empower local patients with accessible, medically accurate information regarding their DNA, demystifying the "diagnostic odyssey" and genetic testing.
2. **Software Engineering Rigor:** Apply modern web development standards, including semantic architecture, responsive design, and Web Content Accessibility Guidelines (WCAG).
3. **Academic Integration:** Serve as a foundational portfolio piece for future Master's level research in Genomic Medicine at Swansea University.

## 🛠️ Technical Architecture & Features
This project was developed utilizing clean, native web technologies without heavy frameworks to ensure maximum performance and maintainability.

* **Multi-Page Semantic Structure:** Built with HTML5 semantic tags (`<header>`, `<main>`, `<article>`, `<section>`) to ensure top-tier screen-reader compatibility and SEO.
* **Modern CSS Layouts:** Utilizes native CSS Variables for theme management, alongside CSS Flexbox and Grid for a fully responsive, card-based UI that adapts from desktop to mobile flawlessly.
* **Accessible UI/UX:** Features a soft, approachable "bubbly" design system (large border-radii, diffused shadows) proven to increase patient trust and readability in healthcare applications. High-contrast color palettes (NHS Blue and Genomics Teal) guarantee WCAG compliance.
